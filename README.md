# 英文單字學習器

一個簡潔而強大的英文單字學習工具！

## 功能特色

✨ **智能發音** - 點擊按鈕聽 Google 自然語音  
📚 **自動生成** - 只需輸入英文單字，自動產生翻譯、音標和例句  
💾 **永久保存** - 你的所有單字和學習進度都會被保存  
🎯 **進度追蹤** - 標記已學習的單字，追蹤學習進度  
🔍 **快速搜尋** - 輕鬆搜尋和篩選單字  

## 使用方式

1. 打開 `vocabulary_learner.html`
2. 在「新增單字」欄位輸入英文單字
3. 點擊「新增單字」按鈕
4. 系統會自動產生中文翻譯、音標和例句
5. 點擊「🔊 發音」聽發音
6. 點擊「標記已學」追蹤你的進度
